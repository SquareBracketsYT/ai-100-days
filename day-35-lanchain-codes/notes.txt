windows:

mac:
python3 -m venv langchain_env

Activate the environment
windows user:
langchain_env\Scripts\activate

mac user:
source langchain_env/bin/activate


Install packages:
pip install langchain

Output parser:

LCEL:
LangChain Expression Language
- Declarative way to connect LangChain components into reusable pipelines

Without LCEL:
prompt = prompt_template.invoke(data)
response = model.invoke(propmp)
final = parser.invoke(response)

With LCEL:
chain = prompt | model | parser


Runnable:
- Everything in LCEL is a Runnable
